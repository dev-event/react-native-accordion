## Installation

<p>Expo:</p>

```bash
$ expo install react-native-reanimated@2.0.0-rc.0
```


<p>React Native (0.60+):</p>


```bash
$ yarn add react-native-reanimated@2
```
and package:

```bash
$ yarn add react-native-accordion
```
